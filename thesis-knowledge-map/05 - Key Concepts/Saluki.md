# Saluki

**Citation:** Gotovchits et al., 2018
**Role in your thesis:** Comparison target in [[Chapter 2 - Related Work]]

Extremely fast binary analysis (micro-execution of binary paths), but requires vulnerability descriptions in a custom specification language. This limits who can actually use it; you need to know the spec language to write new vulnerability patterns.

[[Operation Mango]] doesn't require custom specs; it works with standard sink/source definitions.
