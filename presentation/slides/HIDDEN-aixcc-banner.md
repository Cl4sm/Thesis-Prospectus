DARPA's Autonomous Cyber Reasoning System competition. The challenge: build a fully autonomous system that discovers zero-day vulnerabilities, analyzes root causes, and patches them without human intervention. Seven real open-source targets, each with hidden bugs. Team Shellphish entered both semifinals and finals, running Artiphishell for 143 continuous hours. The system discovered seven zero-day vulnerabilities, reported them responsibly, and demonstrated that multi-technique coordination produces real security value.
