An AI-Powered CRS for Automatic Vulnerability Identification and Patching

DEF CON 2025 · DARPA AIxCC · Team Shellphish

???

Mango showed we can scale detection to large codebases. We can identify thousands of potential vulnerabilities in real software. But scaling detection without validation produces noise. The next question is whether we can combine multiple analysis techniques to produce stronger evidence for the candidates that matter most. That's what Artiphishell addresses. Instead of running independent tools in isolation, we coordinate fuzzing, static analysis, and machine learning through a shared reasoning framework. This is the second encoding of practitioner knowledge: how experienced security teams orchestrate complementary techniques to cross-reference findings.
